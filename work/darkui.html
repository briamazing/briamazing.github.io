Night Shift. Now for Mac.

As OS X and iOS march forward, the features in each update get closer and closer in similarity. When El Capitan was announced, a little used feature was added: a dark interface setting. Since then there has been endless discussion about how Apple should expand that mode to extend it past just the menu bar and dock. With the pending release of iOS 9's Night Shift feature, I decided to explore what the mac could look like if dark mode extended to the UI of the stock applications.
